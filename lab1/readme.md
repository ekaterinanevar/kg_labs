**Лабораторная работа №1**
**Разработка приложения для конвертации между цветовыми моделями**

**Цель работы:** Создание графического приложения на Java для конвертации цветов между моделями RGB, HSL и CMYK с использованием библиотеки Swing.

**Описание программы**

Программа **ColorConverterApp** представляет собой графическое приложение, реализующее преобразование цветов между тремя цветовыми моделями: **RGB** (Красный, Зеленый, Синий), **HSL** (Тон, Насыщенность, Яркость) и **CMYK** (Голубой, Пурпурный, Желтый, Черный). Интерфейс приложения предоставляет удобный способ просмотра и редактирования цветов с использованием слайдеров и текстовых полей ввода.

**Функциональность**

1. **Конвертация между цветовыми моделями:**
   - **RGB** - компоненты R, G, B в диапазоне 0-255
   - **HSL** - компоненты H (0-360°), S (0-100%), L (0-100%)
   - **CMYK** - компоненты C, M, Y, K в диапазоне 0-100%

2. **Элементы управления:**
   - Три панели с слайдерами и текстовыми полями для каждой цветовой модели
   - Визуальная панель предпросмотра текущего цвета
   - Синхронизированное отображение значений во всех моделях

3. **Особенности реализации:**
   - Автоматическое обновление всех компонентов при изменении любой цветовой модели
   - Защита от циклических обновлений через флаг `updating`
   - Валидация вводимых в текстовые поля значений
   - Округление значений для целочисленного представления

**Техническая реализация**

**Интерфейс:**
- Главное окно наследуется от `JFrame`
- Используется `BorderLayout` для компоновки элементов
- Панели цветовых моделей используют `GridLayout`
- Для слайдеров применяются `ChangeListener`, для текстовых полей - `ActionListener`

**Алгоритмы преобразования:**

1. **RGB → HSL:** Вычисление тона, насыщенности и яркости на основе максимальной и минимальной компонент RGB
2. **HSL → RGB:** Преобразование через промежуточные значения с учетом секторов цветового круга
3. **RGB → CMYK:** Расчет через ключевой компонент K и нормализованные значения
4. **CMYK → RGB:** Обратное преобразование с учетом черной составляющей

**Использование приложения:**

1. **Изменение в RGB:** Перемещение слайдеров R, G, B или ввод значений в текстовые поля
2. **Изменение в HSL:** Регулировка тона (H), насыщенности (S) и яркости (L)
3. **Изменение в CMYK:** Настройка компонент голубого, пурпурного, желтого и черного цветов
4. **Визуализация:** Текущий цвет отображается на правой панели предпросмотра

Все изменения автоматически синхронизируются между моделями, обеспечивая согласованное представление цвета в разных системах.

**Заключение:** В ходе лабораторной работы было разработано приложение, демонстрирующее принципы преобразования между различными цветовыми моделями и особенности создания интерактивного графического интерфейса в Java с использованием Swing.
