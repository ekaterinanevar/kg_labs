**Лабораторная работа №2**  
**Локальная пороговая обработка изображений (методы Ниблэка и Сауволы)**

**Цель работы:** Разработка приложения для бинаризации изображений с использованием адаптивных методов локальной пороговой обработки — Ниблэка и Сауволы.

---

### **Описание программы**

Программа **LocalThresholdingLab2** реализует графический интерфейс для применения методов локальной пороговой обработки. Эти методы позволяют эффективно разделять объекты и фон на изображениях с неравномерным освещением или низким контрастом. 

**Основные функции:**
- Загрузка изображений в различных форматах через `ImageIO`
- Применение метода **Ниблэка** для адаптивной бинаризации
- Применение метода **Сауволы** для улучшенной обработки текстур и шумов
- Визуализация результатов в режиме бинарного изображения

---

### **Интерфейс приложения**

1. **Панель управления:**
   - Кнопка **"Открыть изображение"** — загрузка исходного изображения
   - Кнопка **"Метод Ниблэка"** — применение алгоритма Ниблэка
   - Кнопка **"Метод Сауволы"** — применение алгоритма Сауволы

2. **Область отображения:**
   - Центральная панель с поддержкой прокрутки для изображений любого размера
   - Отображение исходного и обработанного изображений

---

### **Алгоритмы обработки**

#### **1. Метод Ниблэка**
- **Формула порога:**  
  `T(x,y) = μ(x,y) + k * σ(x,y)`  
  где:
  - `μ(x,y)` — локальное среднее яркости в окрестности пикселя
  - `σ(x,y)` — стандартное отклонение яркости
  - `k` — коэффициент (обычно отрицательный, например -0.2)

#### **2. Метод Сауволы**
- **Формула порога:**  
  `T(x,y) = μ(x,y) * [1 + k * (σ(x,y)/R - 1)]`  
  где:
  - `R` — динамический диапазон стандартного отклонения (обычно 128)
  - `k` — коэффициент (обычно 0.5)

### **Техническая реализация**

**Используемые технологии:**
- **Java Swing** — для создания графического интерфейса
- **BufferedImage** — для работы с пиксельными данными
- **ImageIO** — для загрузки и сохранения изображений
- **JScrollPane** — для прокрутки больших изображений

**Ключевые параметры:**
- Размер локального окна: **15×15 пикселей**
- Коэффициент Ниблэка: **k = -0.2**
- Коэффициент Сауволы: **k = 0.5**, **R = 128**

### **Процесс работы**

1. **Загрузка изображения:**
   - Нажмите кнопку **"Открыть изображение"**
   - Выберите файл через диалоговое окно
   - Исходное изображение отобразится в интерфейсе

2. **Применение методов:**
   - **Метод Ниблэка:** Эффективен для изображений с плавным изменением освещенности
   - **Метод Сауволы:** Лучше обрабатывает текстуры и устойчив к шумам

3. **Результат:**
   - Полученное бинарное изображение отображается вместо исходного
   - Пиксели выше порога становятся белыми, ниже — черными

---

### **Особенности реализации**

- **Локальная обработка:** Для каждого пикселя вычисляется порог на основе статистики его окрестности
- **Преобразование в градации серого:** Цветные изображения конвертируются используя формулу:  
  `Y = 0.3*R + 0.59*G + 0.11*B`
- **Обработка границ:** Автоматическая обработка краев изображения с частичными окнами
- **Оптимизация:** Прямое вычисление статистики для каждого окна
